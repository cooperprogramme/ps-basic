# PS BASIC PROJECT

Ps Basic is a font testing website created with Vite/React & Tailwind

## Clone

```bash
# clone project
git clone git@github.com:yano1978/ps-basic-font-tool.git ps-basic

# go to project folder
cd ps-basic
```

## Installation

```bash
# install dependencies
yarn

# run project locally
yarn dev

# stop project
ctrl + c

# build project
yarn build
```
